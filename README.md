## Getting Started
<p>kataCode project.</p>

## Prerequisites
1. Node.js (version v19.7.0 or later)
2. TypeScript (version 5.0.2 or later)

## Installation
### Clone the repository:

```
git clone https://github.com/mouadsah/kataCode.git
```

### Install the dependencies:
```
cd kataCode
npm install
```

## Usage

### Run the project:

```
npm start
```

### Run the tests:

```
npm test
```
